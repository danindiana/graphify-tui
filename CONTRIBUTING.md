# Contributing to Graphify TUI

We welcome contributions to Graphify TUI! Here is how you can help:

## Code of Conduct

Please be respectful and constructive in all communications.

## How to Contribute

1. **Fork the Repository**: Clone the repository and create a new feature branch.
2. **Coding Style**:
   - Ensure your code compiles with the latest stable Rust toolchain.
   - Run `cargo fmt` before submitting.
   - Address any warnings shown by `cargo clippy`.
3. **Commit Messages**: Write descriptive, imperative commit messages.
4. **Submit a Pull Request**: Provide a detailed description of your changes and why they are needed.

Thank you for helping improve Graphify TUI!
